# Calculadora Modular Visual

Visualizador interativo de operações em aritmética modular (Z_n), com animações no círculo modular e grafo de conexões.

## Conheça nosso projeto através deste comercial introdutório!

```bash
https://youtu.be/fFbZyzNv_T8
```

## Como rodar

```bash
npm install
npm run dev
```

O servidor abre em `http://localhost:3000`.
